Red Dead Redemption 2 Cheat Codes

Instead of classic cheat codes, RDR2 uses English "code phrases" entered through the in-game menu. Pause > go to Settings, press Triangle (PS4) / Y (Xbox) to open the "Cheats" list, select or type the phrase. Some codes require completing a specific chapter and picking up an in-game newspaper first — noted below.

You can type the phrase directly with a keyboard from the same menu — just grab it with the copy button and paste it.

You'll need to select the phrase letter by letter on a wheel with the controller (no keyboard) — it takes a bit longer, but the same phrases work.
