Fallout 4 Cheat Codes

Cheat codes in Fallout 4 only work on PC and Mac — console commands aren't supported on PS4/PS5/Xbox. Press ~ (tilde/backtick, the key below Esc) to open the console. Some commands (like kill, resurrect) require you to click a character first to select a target.

God Mode & Health

God Mode (Invincibility)
tgm
Immortal Mode (You take damage but can't die)
tim

Movement & Camera

Walk Through Walls Mode (Noclip)
tcl
Free Flying Camera
tfc
Increase Run Speed (e.g. 200 = 2x)
player.setav speedmult 200

Character Progression

Set Level to a Specific Value (e.g. 50)
player.setlevel 50
Increase a S.P.E.C.I.A.L. Stat (e.g. Strength)
player.modav strength 10
Edit Character Appearance
showlooksmenu player 1

Items & Caps

Add 10,000 Caps
player.additem 0000000f 10000
Add a Specific Item to Inventory (enter Item ID and quantity)
player.additem [ItemID] [Quantity]
Teleport to the Test Room With Every Item
coc qasmoke

NPC & Combat Control

Disable NPC AI
tai
Disable Enemy Combat AI
tcai
Undetectable (Stealth)
tdetect
Kill All Nearby Enemies
killall
Kill the Targeted NPC (click first)
kill
Revive the Targeted NPC (click first)
resurrect

Quests

Complete All Main Story Quests
caqs
Restart a Quest (enter Quest ID)
resetquest [QuestID]

Factions

Become an Ally of a Faction
player.AddToFaction [FactionID] 1
Declare a Faction as an Enemy
setenemy [FactionID] 1
Leave a Faction
player.RemoveFromFaction [FactionID]

World & Misc

Reveal the Entire Map
tmm 1
Unlock the Targeted Lock (click first)
unlock
Teleport to a Specific Location (enter Cell ID)
coc [CellID]
Change Character Size
setscale 2
Set Day/Night Cycle Speed
set timescale to 10

Item/NPC/Faction/Quest IDs are found in in-game wiki resources — each has its own unique code, you'll need to add the correct ID yourself.
