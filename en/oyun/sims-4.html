The Sims 4 Cheat Codes

All codes are typed into the cheat console. Most codes require you to type testingcheats true first — that line is included in the list below, run it first.

While in-game, press Ctrl + Shift + C (Command + Shift + C on Mac), type or paste the code into the box that appears, press Enter.

Press and hold all four shoulder buttons at once (L1+L2+R1+R2 on PS4/PS5, LB+LT+RB+RT on Xbox). Type the code into the box that opens using the on-screen keyboard, press submit.

Getting Started

Enable Test Cheats (needed for most codes)

Disable Test Cheats
testingcheats false
